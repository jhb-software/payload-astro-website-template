# De Fryhof

Small localized marketing site with Payload mounted in Next.js.

## Stack

- Next.js 16
- Payload CMS
- Tailwind CSS v4
- shadcn-style components
- TypeScript 7
- oxlint / oxfmt
- pnpm

## Commands

```sh
pnpm install
pnpm seed
pnpm dev
pnpm check
```

Open `/admin` for Payload.

The public site uses Dutch at `/`, English at `/en`, and German at `/de`.
