# Scope

Build a small marketing and group accommodation CMS for De Fryhof.

## In

- Public marketing pages
- Payload admin at `/admin`
- Payload API under `/api`
- Localized content: Dutch, English, German
- SEO fields per page
- Media uploads
- Layout blocks:
  - hero
  - rich text
  - image and text
  - gallery
  - feature grid
  - room list
  - call to action
  - embed

## Out

- Booking engine
- Payment flow
- Custom form builder
- Search
- Redirect manager
- Blog/news system
- Multi-site support
- CRM or email automation

Booking stays an external widget unless the business later needs owned booking data.

## Rule

Add features only when content editors need them. Prefer one Payload field or block over a plugin or custom service.
